# Deterministic RAG Framework
### High-Reliability Agentic Orchestration for Medical Operations

This framework is a production-grade implementation of the **Zero-Hallucination Pipeline** logic. It is designed to transform noisy clinical data into structured, governed outputs.

## Core Architecture
- **State Governance:** Stateful agents powered by LangGraph.
- **Strict Validation:** Pydantic-based schema enforcement at every node.
- **EvalOps Integration:** Real-time hallucination scoring via RAGAS.
- **Safety Layer:** GDPR-shielding and PII-redaction filters.
